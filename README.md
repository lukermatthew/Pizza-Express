
# Pizza Express.

An express pizza ordering  

Tech used (React Router data loading, Redux, Redux Toolkit, thunks, Tailwind CSS)

git clone git@github.com:lukermatthew/Pizza-Express.git

yarn install
yarn run dev - to run web
yarn run dev --host

- Routing
AppLayout
-Home
-Menu
-Cart
-CreateOrder
-Order
